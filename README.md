# Ansible: pm2-install

Install and configure PM2 for a nodejs project

Available on Ansible Galaxy: [pgkehle.pm2-install](https://galaxy.ansible.com/pgkehle/pm2-install)


## Tags

    all:    run everything
    core:   PM2 core module
    logs:   pm2-logrorate
    
## Examples

```YAML

  - hosts: all
  
    roles:
      - pgkehle.pm2-install
```
